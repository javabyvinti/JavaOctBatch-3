--> identifier --> class name, method name, variable name, constant name, etc.

Rules:
--> no keyword can be used for identifier
--> cannot start with a number
--> can include numbers after character
--> underscore and $ can be used
--> can be of any length

--> constants--> fixed value
	--> keyword

	final 

--> keywords--> pre-defined meaning or special meaning

--> variables--> has a particular type and data/range

	final int value = 998.7535320;

	value1 = value1 + value;

}
